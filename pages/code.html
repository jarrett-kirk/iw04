Coding a LFSR in Python

The Galois Linear Feedback Shift Register (LFSR) implementation is encapsulated in a Python class named LFSR. This class simulates the behavior of a Galois LFSR, a type of shift register with feedback logic commonly used in generating pseudorandom sequences.

Class Initialization and Functions:

Upon instantiation, the LFSR class requires two parameters: bits and taps. The bits parameter represents the initial state of the LFSR, specified as an array of binary digits (0 or 1). The taps parameter defines the positions in the state array to which feedback is applied.
Stepping Through the LFSR:
The step method advances the LFSR by one step. During each step, the state is shifted, and if the output bit is 1, an XOR operation is performed with the bits corresponding to the specified tap positions.
Generating a Sequence:
The generate method generates a sequence of bits by repeatedly calling the step function. The length of the generated sequence is determined by the parameter k.
Cycling Through States:
The cycle method takes the current state of the LFSR and iteratively calls the step function until the LFSR returns to its original state. The method returns the number of steps required to complete a full cycle.
